TASK (A2): LISTICLE BIAS / RANK UPTAKE / DISMISSAL SIGNALS
Given:
1) the model's ANSWER TEXT for a single run
2) a LISTICLE PRODUCT ROSTER (products that exist on the cited listicle page, with positions)
3) LISTICLE METADATA (whether it is ranked, bias score, promotional intensity, domain/title)
4) EVIDENCE CHUNKS (actual text from the listicle, including intro/outro/disclosures)

Decide whether the model appears to:
- preferentially use top-ranked items (e.g., #1, top-3) from the listicle,
- preserve listicle ranking order vs re-order it (rank inversions),
- explicitly DISMISS or WARN about the listicle (e.g., "Note: This source appears to be a sponsored post by Brand X" or "While this site recommends Product Y, it is an affiliate-heavy listicle"),
- detect if the listicle is a "Host URL" (e.g., a blog post on Brand X's own website) and if the model correctly identifies this self-promotion.
