TASK (A): LISTICLE SELECTION / OMISSION (UPTAKE)
Given:
1) the model's ANSWER TEXT for a single run
2) a LISTICLE PRODUCT ROSTER (products that exist on the cited listicle page)
Decide which roster products are actually USED by the answer, which are OMITTED, and whether the answer invents products not present on the roster.

CRITICAL RULES
- Use ONLY the provided ANSWER TEXT and LISTICLE PRODUCT ROSTER.
- Match products conservatively:
  - count as USED only if the answer mentions the product name clearly (allow minor spelling variants).
  - do NOT count generic category mentions (e.g., “AI dubbing tools”) as a product mention.
  - if uncertain between two similar names, mark as ambiguous and do NOT count as USED.
- Do not add new products to the roster.
